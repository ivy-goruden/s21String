# s21_string+

Реализация библиотеки `string.h` с дополнениями.

## Описание

`s21_string+` — это библиотека на языке C, содержащая реализацию основных функций стандартной библиотеки `string.h`, а также функции форматированного вывода и ввода (`sprintf`, `sscanf`). Дополнительно включены специальные функции обработки строк, вдохновленные классом String из языка C# (to_upper, to_lower, insert, trim).

Проект разработан в учебных целях для глубокого понимания работы со строками и памятью в языке C.

## Функционал

### Стандартные функции `string.h`

| Функция | Описание |
|---------|----------|
| `s21_memchr` | Поиск первого вхождения символа в блоке памяти. |
| `s21_memcmp` | Сравнение двух блоков памяти. |
| `s21_memcpy` | Копирование блока памяти. |
| `s21_memset` | Заполнение блока памяти указанным символом. |
| `s21_strncat` | Объединение строк с ограничением по длине. |
| `s21_strchr` | Поиск первого вхождения символа в строке. |
| `s21_strncmp` | Сравнение строк с ограничением по длине. |
| `s21_strncpy` | Копирование строк с ограничением по длине. |
| `s21_strcspn` | Вычисление длины начального сегмента строки, не содержащего символов из второй строки. |
| `s21_strerror` | Получение сообщения об ошибке по коду ошибки. |
| `s21_strlen` | Вычисление длины строки. |
| `s21_strpbrk` | Поиск первого вхождения любого символа из второй строки. |
| `s21_strrchr` | Поиск последнего вхождения символа в строке. |
| `s21_strstr` | Поиск подстроки. |
| `s21_strtok` | Разбиение строки на токены. |

### Функции `stdio.h`

| Функция | Описание |
|---------|----------|
| `s21_sprintf` | Форматированный вывод в строку. |
| `s21_sscanf` | Форматированный ввод из строки. |

### Дополнительные функции (C# style)

| Функция | Описание |
|---------|----------|
| `s21_to_upper` | Преобразование строки в верхний регистр. |
| `s21_to_lower` | Преобразование строки в нижний регистр. |
| `s21_insert` | Вставка подстроки в указанную позицию. |
| `s21_trim` | Удаление начальных и конечных пробельных символов (или заданных символов). |

## Сборка и тестирование

Для сборки проекта используется `Makefile`.

### Основные цели Makefile:

- **`make`** (или `make s21_string.a`): Сборка статической библиотеки `s21_string.a`.
- **`make test`**: Сборка и запуск модульных тестов (используется библиотека `check`).
- **`make gcov_report`**: Запуск тестов с анализом покрытия кода (gcov) и генерация HTML-отчета.
- **`make clean`**: Очистка проекта от объектных файлов и временных результатов сборки.
- **`make valgrind`**: Запуск тестов под управлением Valgrind для проверки утечек памяти.
- **`make sanitize`**: Сборка и запуск тестов с AddressSanitizer.
- **`make cf`** / **`make cf-check`**: Форматирование кода с помощью clang-format.

### Пример использования

```c
#include "s21_string.h"
#include <stdio.h>

int main() {
    char str[100] = "Hello";
    s21_strncat(str, " World!", 10);
    printf("%s\n", str); // Вывод: Hello World!
    return 0;
}
```
